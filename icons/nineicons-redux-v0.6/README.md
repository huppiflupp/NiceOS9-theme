# NineIcons-Redux
The beautiful NineIcons pack by [Grassmunk](https://github.com/grassmunk/Platinum9) but modified for KDE. I’ve included lots of missing icons and improved some of the originals.

Note: This icon theme is only tested on KDE Plasma and is not guarenteed to work on gnome, xfce, etc.

## Installation

1. Copy the NineIcons folder into `~/.local/share/icons` to make icons available for the current user, or into `/usr/share/icons` to make icons available for all users.
2. (optional) Update the icon cache: `$ rm ~/.cache/icon-cache.kcache`
